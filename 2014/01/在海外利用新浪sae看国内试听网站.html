在海外利用新浪SAE看国内试听网站

Network Norckon 2014-01-30 3448℃ 0评论

由于国内的某些因素（其实就是D版资源不想外流），国内多家视听服务网站已经不对海外IP提供此类服务。但是海外游子对国内的这一特色（如优酷，土豆，百度音乐等）还是挺喜欢的，但是苦于在海外无法正常使用（会出现如下图一样的情况），那怎么办呢？

既然互联网上有通过利用谷歌的GAE服务，在上面上传一个小小的程序，就可以通过谷歌的GAE服务器做一个中转，从而可以访问YouTube等网站。那我们可不可以利用新浪的SAE来完成类似的工作呢？答案是肯定的。

言归正传，下面我们来介绍如何使用SAE创建一个代理服务器，并且通过这个代理服务器来上网。下面是相对详细的操作步骤，有些步骤由于相对较简易，就省略掉了；如果实在不会操作可以参考GAE。

如果你比较纠结这个事情的话，可以用北极光配置好的一个公用代理，不过公用的流量有限：http://pan.baidu.com/s/1bnmZagB

第一步、首先你需要注册一个新浪的账号，然后打开http://sae.sina.com.cn/?m=myapp输入你的基本信息，比如你的姓名或者其他的注册必要的信息，完成本次注册。

第二步、申请成功后，就会进入我的首页界面，此时你需要点击创建新应用，然后弹出如下窗口，按照如下图片设置即可成功。其中appid和应用名称你可以使用自己喜欢的一个名字。

第三步、然后你需要下载goagent软件，下载地址http://code.google.com/p/goagent/。下载完成后请解压他们，你将会得到两个文件夹，local和server文件夹。

第四步、此时再返回SAE我的首页界面，点击你刚刚创建的应用的名字进入管理面板。然后在左边的菜单中找到代码管理，并且点击进入。

第五步、然后在右边，会提示你创建一个新的版本，此时我们点击创建按钮，并且使用默认的参数创建一个新的版本，再返回代码管理界面。

第六步、接下来我们返回代码管理界面，然后我们找到版本1,右侧的操作按钮，点击后再选择编辑代码。就会出现下面的这个界面。然后我们右击index.php删除，再然后点击图中所示的上传按钮，上传server文件夹中php文件夹中的所有文件。

第七步、SAE的工作我们已经做完了，接下来我们应该处理的是goagent的配置了，你可以安装下面的代码进行相应的设置。如果遇到本文没有提及的代码就不要删除或者编辑本身的代码。

[listen]
ip = 127.0.0.1
port = 59210    ;改成 50000 - 60000 之间
visible = 0     ;改成 0 表示不显示代码窗口
debuginfo = 0

[gae]
appid = FC-SYSTEM   ;随便填写一串字符
password =
path = /2
mode = https
profile = ipv4
window = 4
obfuscate = 0
validate = 0
options =

[ｐｈｐ]
enable = 1 ;此处必须改成 1 表示启用 
password = 123456 
crlf = 0 
validate = 0 
listen = 127.0.0.1:8087 
;下面改成SAE代码管理界面的链接地址 
fetchserver = http://1.xxxproxy.sinaapp.com/ 
usehosts = 0 ;此处改成 0

第八步、保存配置文件以后，我们打开goagent.exe主程序，程序会最小化在系统通知区域（右下角），我们按照下图右击后，重启浏览器就可以看优酷、土豆等等了。

本文参考自：http://www.lovelucy.info/sae-china-proxy.html
