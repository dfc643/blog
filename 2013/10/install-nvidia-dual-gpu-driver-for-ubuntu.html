Install nVIDIA dual-GPU driver for Ubuntu

Operating System Norckon 2013-10-19 596℃ 0评论

So many of notebooks are using nVIDIA and Intel Dual-GPU. Now we’re use bumblebee manage nVIDIA GPU. For the first please install bumblebee by the following code.

sudo add-apt-repository ppa:bumblebee/stable
sudo apt-get update
sudo apt-get install bumblebee bumblebee-nvidia

To view the driver is successfully, type following code in Terminal.

lspci | grep VGA
00:02.0 VGA compatible controller: Intel Corporation 3rd Gen Core processor Graphics Controller (rev 09)
01:00.0 VGA compatible controller: NVIDIA Corporation GF108M [GeForce GT 630M] (rev ff)

The first line is the command-line, second line is Intel, thrid line is nVIDIA GPU, “rev ff” means this GPU was disabled.

Now you can using following code open nVIDIA control panel or run a program by nVIDIA gpu.

#nVIDIA Control Panel
sudo optirun nvidia-settings -c :8

#Run a proguam by nVIDIA GPU
sudo optirun PROGRAM
