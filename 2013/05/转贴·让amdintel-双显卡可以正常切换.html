转贴·让AMD/Intel 双显卡可以正常切换

Operating System Norckon 2013-05-06 4282℃ 0评论

博主的话：我为Ubuntu系统安装显卡驱动折腾了很久，找了很多办法，结果都是没有用的，但是发现这个十分管用。按照这个步骤一步步来你的显卡就能够听你的指挥了，步骤十分简单，但是网速好才能速度快。博主在垃圾校园网里面使用用了几个小时才搞定，因为下载速度太慢了。

————–以下为该作者的文章，博主稍作修改，文章尾部将会给出原文地址————–

由于现在有一些笔记本是AMD/Intel双显卡的，而且BIOS里没有禁用集显这个选项，以前用在Uubuntu 12.04中可以安装fglrx闭源驱动，但是只能驱动AMD独立显卡，无法切换到Intel集显下工作，后来升级到Ubuntu 13.04,装上fglrx闭源驱动，发现可以在Intel集显下工作，但是切换到AMD独显之后，开机系统报错 low graphic mode，后来在国外的Ubuntu论坛里发现这一篇文章，按照上面的步骤操作，终于实现了双显卡完美切换，不过还是得注销重登才能生效，下面把步骤说一下。

预安装：
sudo apt-get install build-essential cdbs fakeroot dh-make debhelper debconf libstdc++6
sudo apt-get install dkms libqtgui4 wget execstack libelfg0 dh-modaliases
sudo apt-get install linux-headers-generic xserver-xorg-core libgcc1

若为64位操作系统请继续执行：
sudo apt-get install ia32-libs lib32gcc1 libc6-i386
cd /usr ; sudo ln -svT lib /usr/lib64

下载以下两个文件，放入个人文件夹：
http://www.mirrorservice.org/sites/archive.ubuntu.com/ubuntu//pool/main/u/udev/libudev0_175-0ubuntu13_amd64.deb
https://docs.google.com/file/d/0B0tTaH4qTIIXdDF3NThFWUtrYlU/edit?usp=sharing

删除旧驱动，这一步很重要必须执行：
sudo apt-get remove fglrx*
sudo apt-get remove xserver-xorg-video-intel

安装下载的两个补丁文件：
cd ~
sudo dpkg -i libudev*
sudo dpkg -i xserver-xorg-video-intel_2.21.6-0ubuntu4_amd64.deb

安装AMD官方驱动程序：
下载最新的 ATI Catalyst 官方驱动，可以从官网www.amd.com/drivers下载。解压下载好的zip文件，把后缀为run的文件解压到个人文件夹，然后在终端里面执行（请将XX.X替换成你所下载的版本）：
sudo chmod +x amd-catalyst-XX.X-linux-x86.x86_64.run
sudo sh ./amd-catalyst-XX.X-linux-x86.x86_64.run –buildpkg Ubuntu/raring

不出意外的话，过大约几分钟就会在目录下生成几个deb包了，然后安装这些软件包：
sudo dpkg -i fglrx*.deb

安装之后，在终端下输入，执行第一条的时候可能报错不用理会：
sudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo aticonfig –initial -f

选择使用独显：
sudo aticonfig –px-dgpu

重新启动计算机：
sudo reboot

另外附送一些脚本文件，供大家参考使用：
切换到独显模式：
#!/bin/bash
# Activate discrete GPU (High-Performance mode), must re-start X to take effect
sudo aticonfig –px-dgpu
sudo restart lightdm

切换到集显模式：
#!/bin/bash
# Activate integrated GPU (Power-Saving mode), must re-start X to take effec
sudo aticonfig –px-igpu
sudo restart lightdm

显示当前模式：
#!/bin/bash
#Show current GPU (High- or Low-Performance mode) as notification
aticonfig –pxl | while read SPAM_OUT; do notify-send “$SPAM_OUT”; done
